# 备份说明

## 标准备份

```bash
bash vpsmagic.sh backup --config /opt/vpsmagic/config.env
```

行为：

1. 先做本地备份
2. 打包归档
3. 生成 `.sha256`
4. 如启用远端，则同步上传主目标
5. 如配置异步副本，则再异步复制一份到副本远端

## 仅本地备份

```bash
bash vpsmagic.sh backup --config /opt/vpsmagic/config.env --dest local
```

## 关键结果

正常情况下，归档目录会出现：

- `*.tar.gz`
- `*.tar.gz.sha256`

远端也应看到同名两份文件。
