# 迁移说明

在线迁移用于两台 VPS 都在线的场景。

## 基本用法

```bash
bash vpsmagic.sh migrate root@new-vps
```

## 常见参数

```bash
bash vpsmagic.sh migrate root@new-vps -p 2222 --bwlimit 10m
bash vpsmagic.sh migrate root@new-vps --skip-restore
```

## 建议流程

1. 确认源机业务正常
2. 目标机网络和 SSH 可达
3. 执行迁移
4. 验证恢复摘要和业务链路
5. 最后切换 DNS / 流量入口
