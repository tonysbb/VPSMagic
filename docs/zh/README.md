新用户建议按这个顺序：

1. [零配置起步](./零配置起步.md) — 完成第一次备份和恢复
2. [配置说明](./配置说明.md) — 配置云端远端
3. [恢复说明](./恢复说明.md) — 掌握各种恢复方式

5. [排障说明](./排障说明.md) — 遇到问题时参考

使用 `vpsmagic doctor` 可以快速了解当前机器的业务画像和恢复建议。
