# 能力矩阵

如果你还不知道自己机器属于哪一类，先执行：

```bash
bash vpsmagic.sh doctor
```

再结合本页理解不同模块的恢复等级。

## A 级：自动上线

- Docker Compose
- 反向代理
- Crontab
- 防火墙

## B 级：半自动恢复

- 常见 Systemd 服务
- 用户目录关键配置
- 数据库逻辑导出与文件恢复

## C 级：手动重建

- 独立 Docker 容器
- 强依赖外部凭据且目标机未预置凭据的场景
- 业务层副作用回滚

## 建议使用方式

- A 级模块可以作为主恢复目标
- B 级模块恢复后需要人工确认
- C 级模块应视为“恢复线索”，不是自动拉起承诺
