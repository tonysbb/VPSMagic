# 排障说明

## 远端恢复前置失败

常见原因：

- 缺少 `rclone.conf`
- OCI 缺少 `/root/.oci/config`
- 目标机无法访问远端存储

建议：

1. 优先看“远端恢复前置检查”输出
2. 凭据无法补齐时，先用 `restore --local`

## Compose 服务恢复后不通

优先检查：

- 容器是否全部 `running`
- 端口是否监听
- Compose 出网是否 `ok`
- 反向代理是否 `active`

## Cloudflare / Caddy 证书问题

优先看：

```bash
journalctl -u caddy -n 100 --no-pager
```

如果刚切 DNS，短时间内出现 `525` 并不一定是最终故障，应先确认 ACME 挑战是否已成功。
